# Lead-to-Site Pipeline

**End-to-end personalized cold outreach generator.**

1. Upload a CSV of leads
2. Pipeline scrapes each company's website
3. Generates a personalized branded landing page per lead
4. Deploys each page to Vercel (live URL)
5. Generates a personalized cold outreach email referencing the live site

---

## Quick Start

### Dashboard (Web UI)

```bash
cd /config/lead-to-site-ui
npm install
npm run dev
```

Open http://localhost:3000, add your API keys in Settings, upload a CSV, and run the pipeline.

### CLI

```bash
cd /config/lead-to-site
python3 pipeline.py leads_test.csv --live
```

### CSV Format

Required columns: `company_name`, `email`

Optional columns: `website_url`, `industry`, `location`, `owner_name`, `first_name`, `last_name`, `contact_email`, `phone`, `services` (comma-separated), `brand_color`, `secondary_color`, `tagline`

Example:
```csv
id,company_name,industry,location,website_url,owner_name,first_name,last_name,email,contact_email,phone,services,brand_color,secondary_color,tagline
1,Nova Growth Agency,Digital Marketing,Chicago IL,https://novagrowth.co,Jessica Park,Jessica,Park,jessica@novagrowth.co,hello@novagrowth.co,312-555-0101,SEO,Social Media Marketing,Brand Strategy,#7C3AED,#06B6D4,Growth That Actually Scales
```

## API Keys

| Key | Where to get | Purpose |
|-----|-------------|---------|
| `VERCEL_API_TOKEN` | vercel.com/account/tokens | Deploying landing pages (starts with `vcp_`) |
| `GROQ_API_KEY` | console.groq.com | AI email generation (free tier, Llama 3.3 70B) |
| `DEEPSEEK_API_KEY` | platform.deepseek.com | AI email generation (backup) |

**No API key = template-based email generation still works.**

Tokens are stored in `/tmp/lead-to-site/config.json` (server-side) and read by the pipeline at runtime.

## Architecture

```
leads_test.csv
    ↓
site_scraper.py     → Scrapes company website (Chromium + Crawl4AI fallback)
    ↓
website_template.py  → Generates branded HTML landing page (~20KB per lead)
    ↓
deploy_to_vercel()  → Deploys to Vercel, returns live URL
    ↓
email_generator.py  → Generates cold outreach email (Groq Llama → DeepSeek → Template)
```

### Files

- `pipeline.py` — Main orchestrator. CLI entry point.
- `site_scraper.py` — Website scraping (Chromium system browser, Crawl4AI, Scrapling fallback).
- `website_template.py` — Branded HTML generator with CSS variables, hero/about/services/CTA/footer.
- `email_generator.py` — Cold email generation (Groq Llama / DeepSeek / Template fallback).
- `deepseek_client.py` — DeepSeek API client.
- `leads_test.csv` — 10 sample leads for testing.
- `sample_output/` — Pre-generated HTML files for sample leads.

## Pipeline Output

For each lead:

- **HTML file**: Saved to `/tmp/lead-to-site/{run_id}/{lead_id}.html`
- **Vercel URL**: Live at `https://{slug}-{hash}.vercel.app`
- **Email**: Subject + plain-text body + HTML body, ready to copy or import into SmartLead

## Deployment

### Dashboard (Next.js)

```bash
cd /config/lead-to-site-ui
npm run build
# Deploy to Vercel with token:
vercel --prod --token vcp_...
```

Or push to GitHub and connect to Vercel.

### Vercel Token Type

**Must be `vcp_` token** from `vercel.com/account/tokens` (NOT `vck_` which is a Connect integration token — it only works for read APIs, not deployments).

## Cold Email Output

Each processed lead gets an email with:
- Personalized subject line
- Body mentioning the live Vercel URL (the personalized landing page)
- Industry/service-specific personalization
- Clear low-commitment CTA

Email is generated by Groq Llama 3.3 70B if `GROQ_API_KEY` is set, otherwise DeepSeek, otherwise a template.

Output format:
```json
{
  "subject": "Quick SaaS question",
  "body": "Hi Amanda,\n\nI built a personalized site for CloudScale Pro here: https://cloudscale-pro-123456.vercel.app\n\n...",
  "body_html": "<p>Hi Amanda...</p>",
  "to_email": "amanda@cloudscalepro.com",
  "method": "groq-llama",
  "website_url": "https://cloudscale-pro-123456.vercel.app"
}
```
